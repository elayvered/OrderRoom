# Order Room – Home Tab Hotfix v1.4.3
תיקון מיידי שמבטיח שטאב בית לא יהיה ריק גם אם קבצי האפליקציה לא טוענים תוכן.

מה עושים
1) העלה את הקובץ home.fix.1.4.3.js לשורש הפרויקט ליד index.html
2) ב-index.html הוסף לפני </body>:
<script src="./home.fix.1.4.3.js?v=1.4.3"></script>
3) טען את האתר עם ?v=1.4.3 או בצע רענון קשיח

מה הפאץ׳ עושה
- יוצר או ממלא את #screen-home בתוכן ברירת מחדל אם הוא חסר או ריק
- מסיר hidden מ-#screen-home ומסתיר אחרים
- אם אין ניווט תחתון יוצר אחד בסיסי
- מפעיל activateTab('home') אם קיים או מפעיל מצב ידני
