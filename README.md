# Order Room – מערכת ניהול הזמנות והספקים החכמה של סושירום

Order Room היא אפליקציית ווב קלה ומהירה (HTML/CSS/JS) לניהול הזמנות מספקים, היסטוריית הזמנות, אנליטיקות, וניהול פריטים וספקים – בעברית מלאה ובכיוון RTL.

## פיצ׳רים
- הזמנות מהירות עם הצגת הכמות הקודמת לכל פריט
- שליחה לוואטסאפ בנוסח מוכן
- היסטוריית הזמנות ויומן פעולות
- אנליטיקות חודשיות/טווח תאריכים עם גרף עמודות
- ניהול ספקים ופריטים, כולל עריכה, מחיקה וייצוא/ייבוא JSON
- שמירה מקומית (localStorage) – ללא שרת

## הוראות שימוש (מקומי)
1. הורד/י את הקבצים לכונן
2. פתח/י את `index.html` בדפדפן
3. הנתונים נשמרים אוטומטית ב־localStorage

## פריסה ל־GitHub Pages
1. צור/י רפוזיטורי בשם `order-room`
2. העלה/י את כל הקבצים שבתיקייה הזו
3. ודא/י שקובץ ה־workflow שמצורף פעיל (ראו `.github/workflows/pages.yml`)
4. לאחר ה־push, GitHub Pages ייצור אתר סטטי עם קישור בסגנון:
   `https://<username>.github.io/order-room/`

> לחלופין, ניתן לפרוס ל־Netlify או Vercel על ידי גרירה של התיקייה לאזור הפריסה שלהם.
