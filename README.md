# Order Room – מערכת ניהול הזמנות והספקים החכמה של סושירום

Order Room היא אפליקציית ווב קלה ומהירה (HTML/CSS/JS) לניהול הזמנות מספקים, היסטוריית הזמנות, אנליטיקות, וניהול פריטים וספקים – בעברית מלאה ובכיוון RTL.

## מה יש בגרסה v1.1 (Light)
- מסך בית חדש: יום בשבוע, תאריך, כרטיסי ספקים (לחיצים)
- הזמנות: מילוי כמות לכל פריט, תצוגת כמות קודמת, הערות חופשיות
- שליחה לוואטסאפ/העתקה כנוסח מוכן
- היסטוריה: רשימת הזמנות שנשמרו
- נתונים: גרף עמודות לטווח תאריכים
- הגדרות: ספקים (כולל יום הזמנה), פריטים לכל ספק
- שמירה אוטומטית ב־localStorage (ללא שרת)

## הרצה מקומית
פתח/י את `index.html` בדפדפן. זה הכל.

2. העלה/י את כל קבצי התיקייה
